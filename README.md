# Dungeness Crabs
MSCH-C220 Game Jam, April 2023

## GO ON A GEM HUNT
Scour the maze for the gems!

## Game Play
Collect any gems you see to win.
A and D to move left and right. W to Jump.

## Impediments
Coding the state machine and asset creation was the most difficulty. 

## Implementation
High jumps,
Collectible gems
